Install

Bower

This package is available in bower under the name cinch-charts.

Make sure you have installed bower in your system. If not, follow the steps mention here

To install this package,

Via Bower CLI: 
 bower install cinch-charts

Github

Repository for this project is available in Github. You can see it here.

To clone repository,

Via SSH: 
 git clone git@github.com:bharadhwajcn/cinch-charts.git
Via HTTPS: 
 git clone https://github.com/bharadhwajcn/cinch-charts.git
